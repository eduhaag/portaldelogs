Frontend ativo do projeto:
- fontes de interface em `../frontend-angular/`
- servidor HTTP em `server.js`, nesta pasta

Esta pasta nao contem um app React.
Ela existe apenas para publicar o build do Angular em `frontend-angular/dist/frontend-angular/browser`.

Uso local:
- gerar o build em `../frontend-angular`
- executar `npm start` nesta pasta para servir o build na porta `3000`
